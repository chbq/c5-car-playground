# Package Notes

This repository contains project-owned documentation, automation scripts and a
STM32CubeMX-generated STM32F1 firmware baseline.

Merchant documentation, example archives and other vendor evidence under
`reference/` are intentionally local-only and are not distributed through Git.

Generated CMSIS and STM32 HAL sources come from STM32CubeF1 1.8.7. Their
upstream license files are retained beside the generated sources under
`target/c5-firmware/Drivers/`.
